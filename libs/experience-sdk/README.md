# Futureverse SDK

```
yarn add @futureverse/experience-sdk
```

The Futureverse SDK is a developer toolkit that enables third-party developers
to integrate with Futureverse. The SDK includes an SSO system via a central OIDC
provider, various adapters for popular web3 frameworks such as web3-react and
wagmi, and support for signing messages using custodial wallets managed by
Futureverse.


## Features

- **Single Sign-On:** The SDK includes an SSO system via a central OIDC provider
  that allows users to securely sign in to third-party applications with their
  Futureverse account.
- **Web3 Framework Adapters:** The SDK provides adapters for popular web3
  frameworks such as web3-react and wagmi, making it easy to build web3-powered
  applications that work seamlessly within the Futureverse.
- **Custodial Wallets**: The SDK includes support for signing messages using
  custodial wallets managed by Futureverse, providing a secure and easy-to-use
  signing mechanism for developers.
