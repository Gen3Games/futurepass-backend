# @futureverse/rpc-kit

Build simple and type-safe RPC interfaces with TypeScript and io-ts.

**Features:**

- Client-side cancellation
- Server-side cancellation
- Server->Client events
- Strong types verified at runtime
- Tiny and unobtrusive
- Transport agnostic
- No code-gen
- No boilerplate
- Simple, safe error handling
- Extensible
